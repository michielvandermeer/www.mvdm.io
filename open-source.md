This is an overview of the Open Source packages I've created.

## ASP.Jobs

A simple job scheduling library for ASP.NET applications.

[View on GitHub](https://github.com/mvdmio/ASP.Jobs)

## Hotwire.NET

Brings Hotwire (Turbo and Stimulus) to .NET applications, enabling Rails-like development patterns in ASP.NET.

[View on GitHub](https://github.com/mvdmio/Hotwire.NET)

---

## Database.PgSQL

A PostgreSQL database utility library for .NET applications.

[View on GitHub](https://github.com/mvdmio/Database.PgSQL)

## Tailwind.NET

Adds Tailwind CSS support to .NET projects with easy integration.

[View on GitHub](https://github.com/mvdmio/Tailwind.NET)

## ValueConversion

A C# library for working with units of measurement, making unit conversions simple and type-safe.

[View on GitHub](https://github.com/mvdmio/ValueConversion)
